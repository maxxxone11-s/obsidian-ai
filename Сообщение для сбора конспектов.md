Сделай KNOWLEDGE_EXPORT для моего AI Second Brain.

Контекст:

У меня есть Obsidian Vault, который синхронизируется через Codex.

Это не обычный конспект.

Это пакет знаний для синхронизации существующей базы знаний.

Главная цель:

* обновить существующие концепции;  
* создать только отсутствующие концепции;  
* не создавать дубликаты;  
* сохранить структуру знаний.

⸻

Правила

Не пересказывай чат.

Выделяй только реальные знания.

Не выдумывай темы.

Если концепция уже существовала в обучении, выдай её как обновление.

Если концепция новая — как новую концепцию.

Каждая концепция должна быть самостоятельной.

⸻

Формат:

KNOWLEDGE_EXPORT  
protocol_version: 2.0  
sync_mode: merge  
module:  
  name:  
  knowledge_area:  
  category:  
  source:  
  language: ru  
concepts:  
- topic:  
  aliases:  
  status:  
  confidence:  
  difficulty:  
  summary:  
  simple_explanation:  
  why_it_matters:  
  how_it_works:  
  example:  
  common_mistakes:  
  related:  
  review_questions:  
  next_topics:

В конце обязательно добавь:

learning_state:  
  understood:  
  weak_spots:  
  repeated_mistakes:  
  recommended_next_step:

Не добавляй инструкций для Obsidian.

Не добавляй текста вне KNOWLEDGE_EXPORT.

Не добавляй пояснений.

Верни только готовый KNOWLEDGE_EXPORT.
