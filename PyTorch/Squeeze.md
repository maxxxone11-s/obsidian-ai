# Squeeze

Удаление размерности размера 1 из тензора.

## Простое объяснение

Squeeze удаляет размерность размера 1 из тензора.

## Интуитивное объяснение

Как снятие пустой полки со шкафа — убираем ненужные размерности.

## Зачем это нужно

- Удаление размерности размера 1
- Подготовка данных
- Удаление batch dimension если batch_size=1

## Как это работает

### Операция
- Удаляет размерности размера 1
- Можно указать конкретную размерность

## Пример

```python
import torch

x = torch.randn(1, 3, 1, 4)  # shape: (1, 3, 1, 4)
y = x.squeeze()               # shape: (3, 4)
z = x.squeeze(0)              # shape: (3, 1, 4)
```

## Типичные ошибки

- Squeeze невернай размерности
- Забывают что squeeze может удалить несколько размеров
- Путаница с unsqueeze
