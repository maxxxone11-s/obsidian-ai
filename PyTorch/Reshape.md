# Reshape

Изменение формы тензора с возможным копированием данных.

## Простое объяснение

Reshape изменяет форму тензора, иногда копируя данные, если тензор non-contiguous.

## Интуитивное объяснение

Как пересборка конструктора — разбираем и собираем заново.

## Зачем это нужно

- Безопасное изменение формы
- Работает с non-contiguous tensors
- Более гибко чем view

## Как это работает

### Отличие от view
- Может копировать данные
- Не требует contiguous тензора
- Немного медленнее

## Пример

```python
import torch

x = torch.randn(6)
y = x.reshape(2, 3)  # shape: (2, 3)
z = y.reshape(3, 2)  # можно переформировать
```

## Типичные ошибки

- Используют reshape без необходимости копирования
- Забывают про -1 для автоматического расчёта
- Путаница с view
