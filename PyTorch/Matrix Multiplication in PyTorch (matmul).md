# Matrix Multiplication in PyTorch (matmul)

## Кратко

Изучено различие между поэлементными операциями и матричным умножением. Математическая формула персептрона связана с кодом PyTorch.

## Простое объяснение

`*` умножает элементы по отдельности.

`@` и `torch.matmul` вычисляют скалярное произведение для векторов и матричное умножение для Tensor.

## Зачем это нужно

Любой Linear Layer использует именно матричное умножение.

## Как это работает

- `x * w` — поэлементное умножение.
- `x @ w` — скалярное произведение.
- Формула персептрона: `output = x @ w + b`.
- `matmul` — основа всех [[nn.Linear]].

Персептрону нужен один результат, а не новый вектор. Поэтому используется `@`, а не `*`.

## Пример

```python
import torch

x = torch.tensor([2, 4])
w = torch.tensor([3, 5])

print(x * w)  # tensor([6, 20])
print(x @ w)  # tensor(26)
```

## Типичные ошибки

- Путать `*` и `@`.
- Воспринимать поэлементное и матричное умножение как одинаковые операции.
