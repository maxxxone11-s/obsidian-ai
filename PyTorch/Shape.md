# Shape (Форма тензора)

Размеры тензора в каждом измерении.

## Простое объяснение

Shape — это кортеж, описывающий размеры тензора в каждом измерении.

## Интуитивное объяснение

Как размеры коробки — длина × ширина × высота.

## Зачем это нужно

- Понимание структуры данных
- Отладка моделей
- Правильные операции между тензорами

## Как это работает

### Примеры shape
- (3,) — одномерный вектор длины 3
- (2, 3) — двумерная матрица 2×3
- (32, 3, 224, 224) — batch изображений

## Пример

```python
import torch

x = torch.randn(2, 3, 4)
print(x.shape)  # torch.Size([2, 3, 4])
print(x.size())  # torch.Size([2, 3, 4])
```

## Типичные ошибки

- Неправильный порядок размеров
- Путаница между shape и size
- Забывают про batch dimension
