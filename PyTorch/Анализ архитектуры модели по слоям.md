# Анализ архитектуры модели по слоям

## Простое объяснение

Архитектуру модели можно понимать по последовательности размеров слоев без запуска кода. Размеры Linear показывают движение данных между слоями.

## Зачем это нужно

Такой анализ позволяет быстро понимать чужие PyTorch-проекты и заранее видеть ошибки shape.

## Как это работает

- Следить за shape после каждого слоя.
- Проверять соответствие размеров соседних слоев.
- Анализировать последний слой отдельно.
- Помнить, что [[Neural Networks/ReLU|ReLU]] не меняет shape.

## Пример

```text
architecture:
3 -> 8
8 -> 4
4 -> 2
```

## Типичные ошибки

- Не учитывать изменение shape после Linear.
- Ошибаться в согласовании размеров соседних слоев.
