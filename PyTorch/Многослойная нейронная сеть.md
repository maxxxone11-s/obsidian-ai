# Многослойная нейронная сеть

## Простое объяснение

Первый слой преобразует входные признаки в новые внутренние признаки. Следующий слой использует уже их, а не исходные данные.

## Зачем это нужно

Именно благодаря последовательному построению новых признаков глубокие сети способны решать сложные задачи.

## Как это работает

- Входные признаки поступают в первый слой.
- Первый слой создает новые признаки.
- Следующий слой работает с новыми признаками.
- Последний слой выдает итоговое решение.

## Пример

```text
architecture:
3 -> 5
5 -> 1
```

## Типичные ошибки

- Представлять, что каждый слой снова работает с исходными признаками.

## Вопросы для проверки

- Почему второй слой принимает выход первого?
- Что становится входом второго слоя?
- Почему сеть называется глубокой?
