# Variance

## Кратко

Variance измеряет разброс данных относительно среднего как среднее квадратов отклонений.

## Простое объяснение

Variance показывает, насколько сильно данные разбросаны вокруг среднего.

## Зачем это нужно

Variance позволяет отличать выборки с одинаковым средним, но разным разбросом.

## Как это работает

Для каждого значения вычисляется отклонение от среднего, отклонение возводится в квадрат, затем вычисляется среднее этих квадратов.

```text
σ² = (1 / n) * Σ (x_i - μ)²
```

Где:

- `σ²` — дисперсия.
- `x_i` — значение.
- `μ` — среднее.
- `n` — количество элементов.

## Пример

```text
Данные: [49, 50, 50, 51, 50]
Mean = 50
Квадраты отклонений = [1, 0, 0, 1, 0]
Variance = 0.4
```

## Типичные ошибки

- Усреднять отклонения без возведения в квадрат.
- Не понимать, зачем используются квадраты.
- Путать Variance со [[Standard Deviation]].
