## Импорт KNOWLEDGE_EXPORT — 2026-06-26

**Кратко:** Batch — небольшая часть Train Set, которая используется за один цикл обучения.

**Простое объяснение:** Вместо всего датасета модель берет маленькую пачку данных.

**Зачем это нужно:** После каждого Batch происходит обновление параметров.

**Как это работает:** Один [[Epoch]] состоит из нескольких batch. На каждом batch модель выполняет [[Forward Pass]], считает [[Loss]], делает [[Backpropagation]] и обновляет параметры.

**Пример:** Dataset = 1000 изображений, Batch Size = 100, получаем 10 Batch.
