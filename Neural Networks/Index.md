# 🧠 Neural Networks

## Структура раздела

Нейронные сети и глубокое обучение.

### Основы
- Perceptron
- Multi-layer Perceptron (MLP)
- Активационные функции
- Backpropagation

### Архитектуры
- CNN (Convolutional Neural Networks)
- RNN (Recurrent Neural Networks)
- LSTM и GRU
- Transformer
- Attention механизм

### Обучение
- Оптимизаторы (SGD, Adam)
- Функции потерь
- Регуляризация (Dropout, BatchNorm)
- Transfer Learning

### Приложения
- Компьютерное зрение
- Обработка естественного языка
- Временные ряды
