## Импорт KNOWLEDGE_EXPORT — 2026-06-26

**Кратко:** Test Set используется для окончательной проверки модели после завершения обучения.

**Простое объяснение:** Это честный экзамен. Модель никогда не должна учиться на этих данных.

**Зачем это нужно:** Только Test показывает, насколько хорошо модель умеет работать с совершенно новыми примерами.

**Как это работает:** Test Set используют после завершения выбора модели и гиперпараметров. Частая проверка на test превращает его в validation и портит честную оценку.

**Пример:** Finished Training → Test Set → Final Accuracy.

**Типичные ошибки:**
- Использовать Test Set во время настройки модели.
- Проверять Test после каждой Epoch.

**Вопросы для проверки:**
- Почему Test используют только после обучения?
