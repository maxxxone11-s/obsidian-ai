# Weights (Веса)

Параметры нейронной сети, которые обучаются.

## Простое объяснение

Weights — это числа, которые модель корректирует во время обучения, чтобы улучшить предсказания.

## Интуитивное объяснение

Как регулировка громкости разных инструментов в оркестре — каждый вес определяет, насколько важна каждая входная переменная.

## Зачем это нужно

- Основные параметры модели
- Содержат выученные закономерности
- Их значения определяют поведение сети

## Как это работает

### Инициализация
- Xavier initialization
- He initialization
- Random initialization

### Обновление
- Через gradient descent
- Используя backpropagation

## Пример

```python
import torch
import torch.nn as nn

model = nn.Linear(10, 5)
print(model.weight.shape)  # (5, 10)
```

## Типичные ошибки

- Неправильная инициализация
- Слишком большие значения (>1)
- Не используют регуляризацию для больших весов

## Импорт KNOWLEDGE_EXPORT — 2026-06-26

**Кратко:** Weight — обучаемый параметр модели, определяющий силу влияния каждого признака.

**Простое объяснение:** Если [[Feature]] — это информация, то Weight показывает, насколько сильно этой информации доверяет модель.

**Зачем это нужно:** Именно Weight обучается во время [[Gradient Descent]].

**Как это работает:** Каждый входной признак умножается на свой вес. Во время обучения веса меняются так, чтобы ошибка модели становилась меньше.

**Пример:** Feature = площадь квартиры. Weight = 0.1 — площадь почти не влияет. Weight = 5 — площадь становится очень важной.

**Типичные ошибки:**
- Путать Weight с Feature.
- Считать, что Weight хранит данные.
