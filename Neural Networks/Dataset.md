# Dataset

## Простое объяснение
Dataset — это полный набор примеров, который затем делится на Train, Validation и Test.

## Зачем это нужно
Без Dataset модель не сможет найти закономерности. Качество данных напрямую влияет на качество обучения.

## Как это работает
Dataset делят на [[Train Set]], [[Validation|Validation Set]] и [[Test Set]], чтобы отдельно обучать модель, настраивать её и честно проверять финальное качество.

## Пример
Dataset: 10000 изображений кошек и собак → деление на Train / Validation / Test.
