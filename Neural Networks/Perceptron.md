# Perceptron

Простейший элемент нейронной сети.

## Интуитивное объяснение

Как рецепторы в мозгу — получают сигналы, обрабатывают и выдают результат.

## Кратко

Персептрон — простейшая модель искусственного нейрона. Он вычисляет Score и принимает решение.

## Простое объяснение

Perceptron — это модель нейрона, которая принимает входы, применяет веса и выходит результат.

Это первый искусственный нейрон, из которого выросли современные нейросети.

## Зачем это нужно

- Основа нейронных сетей
- Простейшая модель классификации
- Историческое значение

Персептрон позволяет понять основу работы любого нейрона.

## Как это работает

### Формула
y = activation(w·x + b)

Где:
- w — веса
- x — входы
- b — смещение (bias)

Персептрон умножает [[Feature]] на [[Weights|Weight]], добавляет [[Bias]], получает [[Score]] и сравнивает его с порогом.

## Пример

```python
import numpy as np

def perceptron(x, w, b):
    return np.sign(np.dot(w, x) + b)
```

Если Score > Threshold, Output = 1. Иначе Output = 0.

## Типичные ошибки

- Используют perceptron для XOR задачи
- Не нормализуют входы
- Неправильно инициализируют веса

- Путать Perceptron и современный нейрон с функцией активации.
