## Импорт KNOWLEDGE_EXPORT — 2026-06-26

**Кратко:** Epoch — один полный проход по всему Train Set.

**Простое объяснение:** Если модель увидела все обучающие данные один раз, значит закончилась одна Epoch.

**Зачем это нужно:** Epoch используется для оценки прогресса обучения.

**Как это работает:** Внутри одной epoch модель проходит все batch из [[Train Set]]. Параметры обновляются после batch, а не только после epoch.

**Пример:** Dataset = 1000, Batch = 100, Epoch = 10 Batch.

**Типичные ошибки:**
- Считать, что веса обновляются после Epoch.

**Вопросы для проверки:**
