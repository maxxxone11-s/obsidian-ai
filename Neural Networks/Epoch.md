# Epoch

## Кратко

Epoch — один полный проход по всему Train Set.

## Простое объяснение

Stub-заметка для следующего шага после импорта [[Forward Pass]] и [[Backpropagation]].

Если модель увидела все обучающие данные один раз, значит закончилась одна Epoch.

## Зачем это нужно

Эта тема нужна для понимания организации цикла обучения нейросети.

Epoch используется для оценки прогресса обучения.

## Как это работает

Пока не раскрыто.

Внутри одной epoch модель проходит все batch из [[Train Set]]. Параметры обновляются после batch, а не только после epoch.

## Пример

Пока не добавлен.

Dataset = 1000, Batch = 100, Epoch = 10 Batch.

## Типичные ошибки

- Пока не добавлены.

- Считать, что веса обновляются после Epoch.
