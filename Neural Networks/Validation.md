# Validation

## Кратко

Validation Set используется для проверки качества модели во время обучения.

## Простое объяснение

Stub-заметка для связанной темы из импорта [[Optimizer]] и [[Batch]].

Это промежуточная проверка. На ней веса уже НЕ обновляются.

## Зачем это нужно

Эта тема помогает понять обучение, регуляризацию и оценку нейросетей.

Позволяет вовремя заметить переобучение и подобрать гиперпараметры.

## Как это работает

Пока не раскрыто.

После эпохи или нескольких шагов модель проверяют на validation данных. [[Optimizer]] на этих данных не делает update, но результаты помогают менять настройки вроде [[Learning Rate]].

## Пример

Пока не добавлен.

Train → Validation → изменение Learning Rate.

## Типичные ошибки

- Пока не добавлены.

- Путать Validation и Test.
- Думать, что на Validation работает Optimizer.
