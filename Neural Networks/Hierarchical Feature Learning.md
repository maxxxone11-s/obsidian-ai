# Hierarchical Feature Learning

## Простое объяснение

Нейросеть не видит сразу кошку: она постепенно собирает пиксели → линии → формы → глаза → морда → кошка.

## Зачем это нужно

Именно эта идея делает Deep Learning настолько мощным.

## Как это работает

Каждый [[Layer]] в глубокой сети преобразует предыдущие признаки во всё более абстрактные. Ранние слои ловят простые паттерны, поздние — сложные структуры.

## Пример

Image → Edges → Shapes → Object Parts → Object.

## Типичные ошибки

- Думать, что все слои ищут одинаковые признаки.

## Вопросы для проверки

- Почему глубокая сеть лучше одного нейрона?
- Что делает каждый слой?
