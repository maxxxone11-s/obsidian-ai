# ReLU

## Простое объяснение

После Linear ReLU заменяет все отрицательные значения на ноль и передает дальше только активные признаки.

## Зачем это нужно

ReLU добавляет нелинейность между слоями. Без функций активации несколько Linear-слоев эквивалентны одному Linear.

## Как это работает

ReLU вычисляет `max(0, score)`. После первого слоя вычисляются признаки, ReLU оставляет положительные значения, а следующий слой работает с обновленными признаками. Shape при этом не меняется.

## Пример

```text
input:  -3, 8, -1, 5
output:  0, 8,  0, 5
```

## Типичные ошибки

- Думать, что ReLU нормализует положительные значения.
- Воспринимать ReLU как изменение shape.
- Не понимать, зачем обнуляются отрицательные значения.
