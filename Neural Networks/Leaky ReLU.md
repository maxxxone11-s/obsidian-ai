# Leaky ReLU

## Простое объяснение

Вместо полного отключения нейрон продолжает немного работать.

## Зачем это нужно

Leaky ReLU уменьшает вероятность появления Dead Neuron.

## Как это работает

Для положительного score функция ведёт себя как [[ReLU]], а для отрицательного возвращает маленькое отрицательное значение.

## Пример

Если score = -10, output может быть -0.1.

## Типичные ошибки

- Думать, что отрицательные значения полностью сохраняются.
