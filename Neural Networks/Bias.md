# Bias (Смещение)

Параметр смещения в нейронной сети.

## Интуитивное объяснение

Как переднее смещение перпендикуляра — даёт свободу вращения.

## Кратко

Bias — дополнительный обучаемый параметр, позволяющий смещать решение нейрона.

## Простое объяснение

Bias — это дополнительный параметр, который позволяет модели смещать предсказания.

Bias позволяет нейрону изменить решение, даже если входные данные одинаковые.

## Зачем это нужно

- Позволяет модели смещать предсказания
- Увеличивает выразительность сети
- Улучшает обучение

Без Bias нейрон становится значительно менее гибким.

## Как это работает

### Формула
y = activation(w·x + b)

b — это bias (смещение)

### Особенности
- Обычно один bias на нейрон
- Инициализируют нулями
- Обновляют при обучении

Bias добавляется к сумме feature × weight и сдвигает [[Score]] до применения [[Activation Function]].

## Пример

```python
import torch.nn as nn

layer = nn.Linear(10, 5, bias=True)
print(layer.bias.shape)  # (5,)
```

`score = feature * weight + bias`

## Типичные ошибки

- Забывают добавить bias
- Инициализируют не нулями
- Отключают bias без необходимости

- Думать, что обучаются только Weight.
