# Bias (Смещение)

Параметр смещения в нейронной сети.

## Простое объяснение

Bias — это дополнительный параметр, который позволяет модели смещать предсказания.

## Интуитивное объяснение

Как переднее смещение перпендикуляра — даёт свободу вращения.

## Зачем это нужно

- Позволяет модели смещать предсказания
- Увеличивает выразительность сети
- Улучшает обучение

## Как это работает

### Формула
y = activation(w·x + b)

b — это bias (смещение)

### Особенности
- Обычно один bias на нейрон
- Инициализируют нулями
- Обновляют при обучении

## Пример

```python
import torch.nn as nn

layer = nn.Linear(10, 5, bias=True)
print(layer.bias.shape)  # (5,)
```

## Типичные ошибки

- Забывают добавить bias
- Инициализируют не нулями
- Отключают bias без необходимости

**Кратко:** Bias — дополнительный обучаемый параметр, позволяющий смещать решение нейрона.

**Простое объяснение:** Bias позволяет нейрону изменить решение, даже если входные данные одинаковые.

**Зачем это нужно:** Без Bias нейрон становится значительно менее гибким.

**Как это работает:** Bias добавляется к сумме feature × weight и сдвигает [[Score]] до применения [[Activation Function]].

**Пример:** `score = feature * weight + bias`
