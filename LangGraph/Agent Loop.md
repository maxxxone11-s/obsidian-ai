# Agent Loop

## Простое объяснение
Agent Loop — это повторяющийся цикл: User → Reasoning → Tool → Observation → Reasoning → Answer.

## Зачем это нужно
Практически любой агент повторяет этот цикл, даже если внутри он реализован через граф.

## Как это работает
Модель анализирует запрос, решает, нужен ли [[Tool]], получает наблюдение после вызова инструмента и продолжает рассуждение до итогового ответа.

## Пример
Пользователь просит найти информацию: агент рассуждает, вызывает инструмент поиска, анализирует результат и отвечает.

## Типичные ошибки
- Считать, что Tool вызывается всегда.
