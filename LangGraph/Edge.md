# Edge

Соединение между узлами в LangGraph.

## Простое объяснение

Edge — это связь между двумя узлами, определяющая поток выполнения.

## Интуитивное объяснение

Как стрелка на диаграмме — указывает направление потока от одного узла к другому.

## Зачем это нужно

- Определяет последовательность выполнения
- Передаёт State между узлами
- Создание логики workflow

## Как это работает

### Типы рёбер
- Простое ребро (от узла A к узлу B)
- Условное ребро (зависит от State)

## Пример

```python
graph.add_edge("node1", "node2")
# State перейдёт от node1 к node2
```

## Типичные ошибки

- Неправильный порядок узлов
- Забывают про start и end узлы
- Циклы без условия выхода
