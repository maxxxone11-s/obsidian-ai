# 🔗 LangGraph

## Структура раздела

Фреймворк LangGraph для построения agent workflows.

### Основы LangGraph
- Граф состояний
- Узлы (nodes)
- Переходы (edges)
- Инструменты (tools)

### Архитектура
- StateGraph
- Циклы и условные переходы
- Обработка ошибок

### Интеграция с LLM
- LangChain интеграция
- Работа с моделями
- Memory и контекст

### Примеры
- Простой agent
- Multi-turn conversation
- Tool-using agent
- Agent с памятью

### Продвинутые темы
- Hierarchical agents
- Sub-graphs
- Custom node types
- Persistence
