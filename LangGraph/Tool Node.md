# Tool Node

Узел для выполнения инструмента/функции в LangGraph.

## Простое объяснение

Tool Node — это специальный узел, который выполняет внешний инструмент или функцию.

## Интуитивное объяснение

Как рука робота, которая берёт нужный инструмент и выполняет действие.

## Зачем это нужно

- Выполнение функций/инструментов
- Интеграция с внешними системами
- Расширение возможностей агента

## Как это работает

### Механизм
- Получает информацию о инструменте из State
- Выполняет инструмент
- Обновляет State с результатом

## Пример

```python
def tool_node(state):
    # Выполняем инструмент
    result = execute_tool(state["tool_call"])
    state["results"].append(result)
    return state

graph.add_node("tools", tool_node)
```

## Типичные ошибки

- Забывают обработать ошибки инструмента
- Неправильное обновление State
- Бесконечное выполнение инструментов
