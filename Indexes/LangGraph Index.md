# 🔗 LangGraph Index

Указатель всех концепций, связанных с LangGraph фреймворком.

## Структура

### Основы LangGraph
- [[State]] — состояние графа
- [[Node]] — узлы в графе
- [[Edge]] — соединения между узлами
- [[Conditional Edge]] — условные переходы
- [[Router]] — маршрутизация
- [[Tool Node]] — узел для выполнения инструментов

### Архитектура
- StateGraph
- Typing и State schema
- Node functions
- Edge functions
- Compilations

### Инструменты (Tools)
- Tool definition
- Tool calling
- Tool binding с моделями
- Dynamic tool calling

### Интеграция с LLM
- LangChain integration
- Model binding
- Streaming responses
- Function calling

### Patterns
- Agent loop pattern
- ReAct (Reasoning + Acting)
- Tool-using pattern
- Multi-turn conversations
- Memory management

### Продвинутые темы
- Hierarchical agents
- Sub-graphs
- Parallel processing
- Error handling
- Persistence и checkpointing

### Примеры
- Simple agent
- Research agent
- Code generation agent
- Customer service agent
- Data analysis agent
