# Рекурсивная проверка палиндрома

## Простое объяснение

Палиндром проверяется сравнением крайних символов. Если первый и последний символ одинаковые, остается проверить середину строки.

## Зачем это нужно

Это первый пример рекурсии, которая возвращает логическое значение. Такой подход важен для задач поиска, проверки условий и обхода структур данных.

## Как это работает

Каждый шаг уменьшает строку с двух сторон:

- если длина строки `0` или `1`, это палиндром;
- если крайние символы разные, это не палиндром;
- иначе проверяем `s[1:-1]`.

## Пример

```python
def is_palindrome(s: str) -> bool:
    if len(s) <= 1:
        return True
    if s[0] != s[-1]:
        return False
    return is_palindrome(s[1:-1])

print(is_palindrome("racecar"))  # True
print(is_palindrome("python"))   # False
```

## Типичные ошибки

- Использовать неверный срез `s[1:-2]`.
- Забывать Base Case для пустой строки.
- Проверять всю строку заново вместо уменьшения задачи.
