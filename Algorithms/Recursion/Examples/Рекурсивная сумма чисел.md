# Рекурсивная сумма чисел

## Простое объяснение
Сумма чисел строится так же, как факториал, только вместо умножения используется сложение: `sum(5) = 5 + sum(4)`.

## Зачем это нужно
Этот пример показывает, что рекурсия применяется не только для умножения. Главное — найти меньшую версию той же задачи.

## Как это работает
Формула:

- `sum(n) = n + sum(n - 1)`;
- `sum(0) = 0`.

Base Case равен `0`, потому что это нейтральное значение для сложения.

## Пример
```python
def sum_to_n(n: int) -> int:
    if n == 0:
        return 0
    return n + sum_to_n(n - 1)


print(sum_to_n(5))  # 15
```

## Типичные ошибки
- Не вернуть результат рекурсивного вызова.
- Выбрать неверный Base Case.
- Не увидеть, что `return` выполняется во время размотки стека.
