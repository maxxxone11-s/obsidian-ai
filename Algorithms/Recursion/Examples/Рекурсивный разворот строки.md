# Рекурсивный разворот строки

## Простое объяснение

Строка разворачивается через разворот подстроки: берем первый символ, разворачиваем остаток строки и ставим первый символ в конец.

Для `"hello"` идея такая: `reverse("ello") + "h"`.

## Зачем это нужно

Это первый хороший пример того, что рекурсия работает не только с числами, но и со структурами данных: строками, списками, деревьями.

## Как это работает

Каждый шаг уменьшает строку на один символ. Когда строка пустая или состоит из одного символа, разворачивать уже нечего.

## Пример

```python
def reverse(s: str) -> str:
    if len(s) <= 1:
        return s
    return reverse(s[1:]) + s[0]

print(reverse("Cat"))  # taC
```

## Типичные ошибки

- Использовать `len(s) == 1` и забывать про пустую строку.
- Ошибаться в срезах, например писать `s[1:-2]` вместо нужной части строки.
- Думать, что символ добавляется сразу, хотя он добавляется при размотке.
