# Подсчет количества узлов дерева

## Кратко

Количество узлов вычисляется как сумма результатов всех поддеревьев плюс текущий узел.

## Простое объяснение

Каждый узел считает себя, затем просит детей посчитать свои поддеревья.

## Зачем это нужно

Это базовый пример рекурсивного агрегирования результатов.

## Как это работает

Базовое значение равно `1`, потому что текущий узел тоже считается. Затем результаты детей суммируются.

## Пример

```python
def count_nodes(node):
    count = 1
    for child in node.children:
        count += count_nodes(child)
    return count
```

## Типичные ошибки

- Увеличивать счетчик на `1` вместо использования результата рекурсии.
- Смешивать обход дерева и вычисление результата.
- Игнорировать возвращаемое значение рекурсивного вызова.

## Вопросы для проверки

- Почему начальное значение равно `1`?
- Почему нельзя писать только `count += 1`?
