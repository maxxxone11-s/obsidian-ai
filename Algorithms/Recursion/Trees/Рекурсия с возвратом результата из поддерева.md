# Рекурсия с возвратом результата из поддерева

## Кратко

Рекурсивный вызов может не только выполнять действия, но и возвращать вычисленный результат родителю.

## Простое объяснение

Родитель делегирует решение поддереву, получает готовый ответ и использует его.

## Зачем это нужно

Именно так решается большинство задач на деревья: подсчет, поиск, вычисление высоты, суммы и другие рекурсивные вычисления.

## Как это работает

Каждый ребенок возвращает собственный результат. Родитель объединяет ответы всех детей.

Главная идея: `dfs(child)` просто выполняет обход, а `result += dfs(child)` использует возвращаемое значение.

## Пример

```python
def count_nodes(node):
    count = 1
    for child in node.children:
        count += count_nodes(child)
    return count
```

## Типичные ошибки

- Рекурсивный вызов выполняется, но его результат игнорируется.
- Пытаться самостоятельно считать детей вместо использования `return`.

- Почему рекурсивный вызов должен возвращать значение?
- Чем отличается `dfs(child)` от `count += dfs(child)`?
