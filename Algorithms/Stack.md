# Stack

## Простое объяснение

Stack — структура LIFO: последним пришёл, первым вышел.

## Зачем это нужно

Используется для скобок, DFS, undo, call stack и монотонных структур.

## Как это работает

Основные операции: push, pop, peek. Последний добавленный элемент извлекается первым.

## Пример

Проверка правильной скобочной последовательности.

## Типичные ошибки

- Путать Stack и Queue.
