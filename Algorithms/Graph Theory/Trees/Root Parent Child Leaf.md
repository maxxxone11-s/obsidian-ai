# Root Parent Child Leaf

## Простое объяснение

Root — верхняя вершина дерева, Parent — родитель, Child — потомок, Leaf — вершина без потомков.

## Зачем это нужно

Эти термины используются практически во всех деревьях, traversal-алгоритмах и задачах на рекурсию.

## Как это работает

Каждая вершина дерева имеет положение относительно других вершин: она может быть корнем, родителем, потомком или листом.

## Пример

Filesystem: корневая папка — root, вложенная папка — child, файл или пустая папка без потомков — leaf.

## Типичные ошибки

- Путать leaf с любой нижней вершиной на рисунке, не проверяя наличие потомков.

## Ментальная модель

Любое дерево можно читать сверху вниз.
