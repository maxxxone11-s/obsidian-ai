# Дерево

## Простое объяснение

Дерево — это граф без циклов, где между двумя вершинами существует только один путь.

## Зачем это нужно

Файловая система, DOM, JSON, XML и AST являются деревьями.

## Как это работает

Дерево имеет иерархическую структуру, не содержит циклов и не позволяет “обойти с другой стороны”.

## Пример

Файловая система или структура проекта.

## Типичные ошибки

- Думать, что любой граф с иерархичным рисунком является деревом.

## Ментальная модель

Дерево — граф, где невозможно обойти с другой стороны.
