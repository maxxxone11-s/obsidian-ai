# Путь

## Простое объяснение
Путь — это маршрут по графу.

## Зачем это нужно
Все алгоритмы поиска работают именно с путями.

## Как это работает
Идти можно только по существующим ребрам. Путь может включать несколько вершин и ребер.

## Пример
A → B → C — путь из A в C через B.

## Типичные ошибки
- Считать, что путь можно строить по несуществующим связям.

## Ментальная модель
Путь — это маршрут только по реальным ребрам.
