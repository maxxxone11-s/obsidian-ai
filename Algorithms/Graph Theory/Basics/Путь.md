# Путь

## Простое объяснение

Путь — это маршрут, по которому можно пройти по графу: последовательность вершин, соединенных существующими ребрами.

## Зачем это нужно

На понятии пути основаны DFS, BFS и большинство алгоритмов поиска.

## Как это работает

Каждая следующая вершина пути должна быть соединена ребром с предыдущей. Если ребра между соседними вершинами нет, такой последовательности пути не существует.

## Пример

`D -> B -> A -> C` — путь, если каждое соседнее звено соединено ребром.

## Типичные ошибки

- Называть путем любую последовательность вершин без проверки существующих ребер.

## Ментальная модель

Путь — это маршрут только по реальным ребрам.
