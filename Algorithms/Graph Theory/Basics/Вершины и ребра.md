# Вершины и ребра

## Простое объяснение

Вершина представляет объект графа. Ребро представляет связь между двумя вершинами: люди — вершины, дружба между людьми — ребра.

## Зачем это нужно

Любой алгоритм работы с графами оперирует именно вершинами и ребрами.

## Как это работает

Вершина может существовать самостоятельно. Ребро всегда зависит от существования двух вершин и не может жить отдельно от них.

## Пример

- User -> User.
- File -> File.
- Commit -> Commit.

## Типичные ошибки

- Считать, что вершина исчезает после удаления всех ребер.

## Ментальная модель

Объекты живут самостоятельно. Связи — нет.

## Вопросы для проверки

- Может ли вершина существовать без ребер?
- Может ли ребро существовать без вершины?
