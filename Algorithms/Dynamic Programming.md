# Dynamic Programming

## Простое объяснение
Dynamic Programming решает задачу через сохранение результатов подзадач.

## Зачем это нужно
Используется для задач оптимизации, подсчёта вариантов, строк, путей и рюкзака.

## Как это работает
Нужно определить state, transition, base cases и порядок вычисления. Можно использовать memoization или tabulation.

## Пример
Fibonacci с memoization.

## Типичные ошибки
- Начинать писать код до определения state и transition.
