# SQL

Язык структурированных запросов для работы с базами данных.

## Простое объяснение

SQL — это язык для создания, чтения, обновления и удаления данных в базах данных.

## Интуитивное объяснение

Как библиотечная система каталогов — SQL помогает найти нужную информацию по определённым критериям.

## Зачем это нужно

- Основа для работы с БД
- Структурированное хранилище данных
- Быстрый доступ к информации

## Как это работает

### Основные операции (CRUD)
- CREATE — создание таблиц
- SELECT — чтение данных
- UPDATE — изменение данных
- DELETE — удаление данных

## Пример

```sql
SELECT * FROM users WHERE age > 18;
```

## Типичные ошибки

- SQL injection атаки
- Неправильное использование JOIN
- Отсутствие индексов на больших таблицах
