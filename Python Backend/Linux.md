# Linux

Операционная система и окружение для разработки и развёртывания.

## Простое объяснение

Linux — это свободная операционная система, где разрабатывают и развёртывают большинство backend приложений.

## Интуитивное объяснение

Как инструментальный цех — нужно знать, как им пользоваться, чтобы эффективно работать.

## Зачем это нужно

- Среда для разработки backend
- Сервера обычно на Linux
- Команды для управления системой
- Bash scripting

## Как это работает

### Основные команды
- ls, cd, mkdir — навигация
- cat, grep — просмотр файлов
- chmod, chown — права доступа
- apt, yum — управление пакетами

## Пример

```bash
ls -la
cd /home/user
mkdir project
chmod +x script.sh
```

## Типичные ошибки

- Неправильные права доступа
- Неправильные пути к файлам
- Запуск от root без необходимости
