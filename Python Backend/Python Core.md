# Python Core

Основные концепции и встроенные типы данных Python.

## Простое объяснение

Python Core — это набор базовых типов данных и операций, которые входят в стандартную библиотеку Python.

## Интуитивное объяснение

Как инструменты в ящике для ремонта — типы данных (строки, списки, словари) — это основной инструмент для любой работы с Python.

## Зачем это нужно

- Основа для всех остальных концепций
- Типы данных используются везде
- Понимание позволяет писать эффективный код

## Как это работает

### Основные типы данных
- int, float, str, bool
- list, tuple, dict, set
- None

### Операции
- Арифметические операции
- Логические операции
- Операции со строками

## Пример

```python
# Основные типы
x = 42  # int
s = "hello"  # str
lst = [1, 2, 3]  # list
dct = {"key": "value"}  # dict
```

## Типичные ошибки

- Путаница между типами данных
- Неправильное использование mutable/immutable типов
- Ошибки индексирования
