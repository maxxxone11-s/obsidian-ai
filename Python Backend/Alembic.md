# Alembic

Инструмент для управления миграциями БД в Python.

## Простое объяснение

Alembic позволяет версионировать и отслеживать изменения схемы БД со временем.

## Интуитивное объяснение

Как система контроля версий Git для БД — можно откатываться на предыдущие версии схемы.

## Зачем это нужно

- Управление изменениями схемы
- Откаты при необходимости
- Воспроизводимые развёртывания
- Командная разработка

## Как это работает

### Основные операции
- init — инициализация
- revision — создание миграции
- upgrade — применение миграций
- downgrade — откат миграций

## Пример

```bash
alembic revision --autogenerate -m "Add user table"
alembic upgrade head
```

## Типичные ошибки

- Не коммитят миграции в git
- Ручные изменения схемы без миграций
- Конфликты при работе в команде
