# Classification

## Простое объяснение

Ответ не число, а класс.

## Зачем это нужно

Это второй основной тип задач ML.

## Как это работает

Модель выбирает один из классов: spam/not spam, fraud/not fraud, cat/dog. Часто результат можно интерпретировать как вероятность класса.

## Пример

Спам / Не спам.

## Типичные ошибки

- Путать Classification и Logistic Regression.
