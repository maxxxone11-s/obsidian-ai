# Loss Function

Функция потерь для измерения ошибки модели.

## Простое объяснение

Loss Function измеряет, насколько предсказания модели отличаются от истинных значений.

## Интуитивное объяснение

Как метр для измерения качества — чем меньше потери, тем лучше модель.

## Зачем это нужно

- Измеряет качество предсказаний
- Используется для оптимизации модели
- Позволяет сравнивать модели

## Как это работает

### Основные типы
- MSE (Mean Squared Error)
- CrossEntropy (для классификации)
- MAE (Mean Absolute Error)

## Пример

```python
import torch.nn as nn

loss_fn = nn.MSELoss()
loss = loss_fn(predictions, targets)
```

## Типичные ошибки

- Выбирают не подходящую функцию потерь
- Не нормализуют выходы перед расчётом потерь
- Забывают про weight параметры

## Импорт KNOWLEDGE_EXPORT — Machine Learning Fundamentals — 2026-06-26

**Кратко:** Loss показывает качество текущего ответа модели. MAE усредняет абсолютную ошибку. MSE возводит ошибку в квадрат.

**Простое объяснение:** Loss показывает, насколько модель ошиблась.

**Зачем это нужно:** Без Loss невозможно обучить модель.

**Как это работает:** Loss превращает ошибку модели в число. MAE считает среднюю абсолютную ошибку, MSE сильнее штрафует большие ошибки за счёт квадрата.

**Пример:** Prediction → Loss → изменение весов.

**Типичные ошибки:**
- Думать, что Loss сам исправляет модель.
