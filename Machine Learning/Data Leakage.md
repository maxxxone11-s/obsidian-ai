# Data Leakage

## Простое объяснение
Модель получает подсказку из будущего.

## Зачем это нужно
Data Leakage делает результаты тестирования нереалистично хорошими.

## Как это работает
Утечка возникает, когда в train или features попадает информация, которая не будет доступна в production. Модель выглядит сильной на тесте, но проваливается на реальных данных.

## Пример
Использование информации через год при прогнозировании сегодняшнего события.

## Типичные ошибки
- Сложно самостоятельно придумывать реальные примеры утечки.
