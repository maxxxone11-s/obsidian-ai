# Confusion Matrix

## Простое объяснение

Confusion Matrix — таблица всех возможных исходов классификации.

## Зачем это нужно

Позволяет понимать ошибки модели.

## Как это работает

Матрица показывает, сколько объектов модель классифицировала правильно и неправильно по каждому классу. Из неё считаются precision, recall и F1.

## Пример

Precision = TP / (TP + FP), Recall = TP / (TP + FN).
