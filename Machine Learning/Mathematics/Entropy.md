# Entropy

## Простое объяснение
Чем равномернее вероятности, тем выше Entropy.

## Зачем это нужно
Используется в LLM и задачах классификации.

## Как это работает
Если модель почти уверена в одном классе, энтропия низкая. Если вероятности распределены равномерно, неопределенность высокая.

## Пример
`50% / 50%` → высокая Entropy.
