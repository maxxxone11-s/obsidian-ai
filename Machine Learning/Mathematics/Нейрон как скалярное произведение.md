# Нейрон как скалярное произведение

## Простое объяснение

Один нейрон = один dot product.

## Зачем это нужно

Это главная связь между математикой и нейросетями.

## Как это работает

Входной вектор признаков умножается на вектор весов через dot product, затем обычно добавляется bias и применяется activation function.

## Пример

`output = X · W`.
