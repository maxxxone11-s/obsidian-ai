# Вектор

## Простое объяснение

Один объект превращается в набор признаков.

## Зачем это нужно

Почти все данные в ML представляются векторами.

## Как это работает

Каждый элемент вектора обычно соответствует признаку объекта. Например, рост, вес и возраст можно хранить как один числовой вектор.

## Пример

`[180, 80, 25]`
