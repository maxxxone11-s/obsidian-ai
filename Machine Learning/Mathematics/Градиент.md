# Градиент

## Простое объяснение
Инструкция, как менять каждый вес.

## Зачем это нужно
Это основа Gradient Descent.

## Как это работает
Градиент собирает частные производные по всем параметрам модели и показывает направление самого быстрого роста Loss. Для уменьшения Loss идут в противоположную сторону.

## Пример
`[10, 5, -2]`.

## Типичные ошибки
- Считать градиент одним числом.
