# Производная

## Кратко

Повторены только те производные, которые используются в обучении нейросетей.

## Простое объяснение

Производная показывает, насколько изменится результат при небольшом изменении входа.

## Зачем это нужно

Это основа обучения моделей. Именно производные вычисляет [[PyTorch/Autograd|Autograd]].

## Как это работает

В обучении производная помогает понять, как изменение параметра влияет на Loss.

Базовые правила:

- `d(c)/dx = 0`
- `d(x)/dx = 1`
- `d(ax)/dx = a`
- `d(x²)/dx = 2x`
- `d(x³)/dx = 3x²`

Производная отвечает на вопрос: что произойдёт с результатом, если немного изменить вход?

## Пример

```text
y = x²
dy/dx = 2x
```

Изменили вес → изменился Loss.

## Типичные ошибки

- Путать изменение входа и изменение Loss.
- Забыть подставить значение `x` после нахождения производной.
- Не до конца понимать физический смысл производной.
