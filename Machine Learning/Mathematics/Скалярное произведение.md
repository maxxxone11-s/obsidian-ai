# Скалярное произведение

## Простое объяснение

Перемножаем элементы и складываем.

## Зачем это нужно

Это основа матричного умножения и работы нейронов.

## Как это работает

Для двух векторов одинаковой размерности перемножаются соответствующие элементы, затем результаты складываются.

## Пример

`[1, 2] · [3, 4] = 11`.
