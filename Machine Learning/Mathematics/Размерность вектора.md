# Размерность вектора

## Простое объяснение
Размерность показывает, сколько признаков содержит объект.

## Зачем это нужно
Размерность определяет форму данных.

## Как это работает
Если вектор содержит три числа, его размерность равна 3. В ML размерность должна совпадать с количеством признаков, ожидаемых моделью.

## Пример
`[1, 2, 3]` → размерность 3.
