# Частная производная

## Простое объяснение

Меняем один вес, остальные фиксированы.

## Зачем это нужно

Именно так вычисляется вклад каждого веса.

## Как это работает

Когда параметров много, частная производная показывает влияние одного выбранного параметра при фиксированных остальных.

## Пример

`∂Loss/∂w1`.

## Типичные ошибки

- Путать параметр функции и элемент вектора.

## Вопросы для проверки

- Что показывает ∂Loss/∂w?
