# Что такое Machine Learning

## Простое объяснение

В обычном программировании человек пишет правила. В ML правила ищет сама модель.

## Зачем это нужно

Это фундамент всей области AI.

## Как это работает

Модель получает признаки, обучается на примерах и подбирает функцию, которая по входу делает прогноз. Обычно модель ищет корреляции в данных, а не настоящие причинно-следственные связи.

## Пример

Features → Model → Prediction.

## Типичные ошибки

- Путать Machine Learning с Deep Learning.
- Считать, что модель понимает причинно-следственные связи, хотя чаще она ищет корреляции.
