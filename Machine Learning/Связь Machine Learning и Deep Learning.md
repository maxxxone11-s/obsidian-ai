# Связь Machine Learning и Deep Learning

## Простое объяснение
В классическом ML признаки часто создает человек. В Deep Learning признаки постепенно строит сама сеть.

## Зачем это нужно
Формирует мост к изучению нейронных сетей.

## Как это работает
Machine Learning — широкая область. Deep Learning входит в неё и использует многослойные neural networks для автоматического извлечения признаков.

## Пример
ML ⊃ Deep Learning.

## Типичные ошибки
- Сначала казалось, что Machine Learning и Deep Learning — разные области.
