# Logistic Regression

Алгоритм для бинарной классификации. Использует градиентный спуск, поэтому требует масштабирования.

## Кратко

Logistic Regression используется для Classification, несмотря на название. Предсказывает вероятность принадлежности к классу.

## Простое объяснение

Выход всегда вероятность от 0 до 1.

Logistic Regression предсказывает вероятность класса, несмотря на слово Regression в названии.

## Зачем это нужно

Это базовый алгоритм бинарной классификации.

Это базовый и важный алгоритм бинарной классификации.

## Как это работает

Logistic Regression строит линейную комбинацию признаков и преобразует её в вероятность класса через sigmoid.
