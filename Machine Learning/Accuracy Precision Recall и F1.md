# Accuracy, Precision, Recall и F1

## Простое объяснение

Accuracy может обманывать. Precision отвечает за качество найденного. Recall — за полноту поиска.

## Зачем это нужно

Это главные метрики классификации.

## Как это работает

Метрики строятся на TP, FP, TN и FN из confusion matrix. Precision важен, когда дорог ложный positive, Recall — когда опасно пропустить нужный объект.

## Пример

Поиск рака, поиск мошенничества.

## Типичные ошибки

- Изначально путались формулы Precision и Recall.
- Accuracy сначала воспринималась как универсальная метрика.
