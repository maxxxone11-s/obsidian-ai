# Cross Validation

## Простое объяснение
Один Test может случайно оказаться слишком простым.

## Зачем это нужно
Позволяет получить более честную оценку качества.

## Как это работает
Данные делятся на несколько folds. Модель обучается и проверяется несколько раз, каждый раз на другом validation fold.

## Пример
K-Fold Cross Validation.
