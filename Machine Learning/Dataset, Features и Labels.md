# Dataset, Features и Labels

## Простое объяснение

Feature — информация, по которой модель делает вывод. Label — правильный ответ.

## Зачем это нужно

Без понимания структуры данных невозможно обучать модели.

## Как это работает

В supervised learning датасет содержит строки-объекты, признаки для описания объектов и label как правильный ответ, который модель учится предсказывать.

## Пример

Возраст, доход → вернет кредит?

## Типичные ошибки

- Путать Feature и Label.

## Вопросы для проверки

- Что такое Feature?
- Что такое Label?
