# Regression

## Простое объяснение
Если ответ — число, значит это Regression.

## Зачем это нужно
Это один из двух основных типов задач ML.

## Как это работает
Модель получает признаки объекта и предсказывает непрерывное числовое значение: цену, температуру, спрос или зарплату.

## Пример
Цена квартиры, зарплата, температура.

## Типичные ошибки
- Путать Regression с алгоритмом Linear Regression.
